# Deep Sleep app

본 프로젝트는 수면 유도를 위한 4-7-8 호흡 가이드 및 수면 사운드를 제공하는 웹 애플리케이션이다. 
시중에 출시된 유료 수면 앱의 무분별한 결제 유도에 불편함을 느껴, 개인적인 목적으로 백그라운드 재생이 완벽히 지원되는 무료 앱을 직접 개발하게 되었다.

## 🛠 주요 구현 기술 및 기술 스택 (Tech Stack)

본 애플리케이션은 순수 바닐라 웹 기술(HTML, CSS, JS)만으로 네이티브 앱 수준의 오디오 제어와 백그라운드 환경을 구현하는 데 집중하였다.

### 1. 오디오 엔진 (Web Audio API & AudioWorklet)
HTML5 `<audio>` 태그의 고질적인 루프 간극(Gap, 끊김 현상) 문제를 완벽하게 해결하기 위해 저수준 오디오 제어를 구현하였다.

* **AudioWorklet 실시간 노이즈 생성기**: 
  핑크 노이즈와 브라운 노이즈를 재생할 때, 미리 녹음된 오디오 파일을 반복(Loop)하는 방식을 사용하지 않는다. 대신 `AudioWorkletProcessor`를 사용하여 수학적 알고리즘을 통해 **실시간으로 오디오 샘플을 무한히 생성**한다. 이를 통해 루프 경계 자체가 존재하지 않는 완벽한 무손실 백색 소음을 구현하였다.
* **이중 오실레이터(OscillatorNode) 기반 바이노럴 비트**: 
  델타파(1Hz, 3Hz) 및 세타파(6Hz) 유도를 위해, 양쪽 귀(채널)에 각기 다른 주파수의 사인파(Sine Wave)를 `OscillatorNode`로 발생시킨다. `ChannelMergerNode`를 활용해 좌우 채널을 분리하고 뇌파 동조 효과를 극대화하였다.
* **원활한 크로스페이드 트랜지션**: 
  `GainNode`의 `linearRampToValueAtTime` 메서드를 활용하여, 사운드 전환 시 볼륨이 튀는 현상을 막고 페이드 인/아웃(Fade in/out) 처리를 통해 자연스럽게 이어지도록 설계하였다.

### 2. 백그라운드 재생 및 미디어 제어
모바일 브라우저(특히 iOS Safari)의 엄격한 오디오 백그라운드 중지 정책을 우회하고 사용자 경험을 높이기 위한 기법들을 적용하였다.

* **Silent WAV Hack (무음 오디오 기법)**: 
  화면이 꺼지거나 앱이 백그라운드로 전환될 때 Web Audio API 컨텍스트가 정지되는 것을 막기 위해, 보이지 않는 `<audio>` 태그로 무음(Silent) WAV 파일을 백그라운드에서 무한 루프 재생하도록 하여 미디어 세션을 강제로 유지시켰다.
* **Wake Lock API**: 
  사용자가 호흡 가이드 등을 볼 때 화면이 자동으로 꺼지는 것을 방지하기 위해 `navigator.wakeLock.request('screen')`을 사용하여 화면 켜짐 상태를 유지한다.
* **Media Session API 연동**: 
  모바일 기기의 잠금 화면이나 알림창에서 현재 재생 중인 수면 사운드의 정보(이름, 아티스트 등)를 확인하고 제어할 수 있도록 메타데이터를 연동하였다.

### 3. PWA (Progressive Web App)
별도의 앱스토어 심사나 설치 없이도 네이티브 앱과 동일한 접근성을 제공한다.

* **Web App Manifest (`manifest.json`)**: 스플래시 화면, 홈 화면 아이콘, 테마 색상, 독립형(Standalone) 디스플레이 모드를 설정하여 브라우저 UI 없이 전체화면으로 구동된다.
* **Service Worker (`sw.js`)**: 오프라인 상태에서도 앱을 실행할 수 있도록 주요 리소스(HTML, JS, CSS)를 캐싱한다.
