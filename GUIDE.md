# Deep_Sleep_app - 사용자 가이드

본 문서는 `Deep_Sleep_app`의 상세한 기기별 사용 방법과 수면 설정 안내를 다룹니다.

## 1. 기기별 실행 및 홈 화면 추가 방법 (설치)

본 애플리케이션은 PWA(Progressive Web App) 기술이 적용되어 있어, 별도의 앱스토어를 거치지 않고 브라우저에서 바로 홈 화면에 추가하여 네이티브 앱처럼 사용할 수 있습니다.

### iOS (iPhone / iPad) 사용자
1. **Safari 브라우저**를 열고 [수면 앱 바로 가기](https://rbtjd215.github.io/Deep_Sleep/)로 이동합니다. (크롬 등 타 브라우저에서는 홈 화면 추가가 제한될 수 있습니다.)
2. 브라우저 하단의 **공유 버튼(네모 상자 위로 화살표가 있는 아이콘)**을 누릅니다.
3. 나타나는 메뉴에서 **'홈 화면에 추가'**를 선택합니다.
4. 우측 상단의 '추가' 버튼을 누르면 기기 바탕화면에 전용 아이콘이 생성됩니다.
5. 이후에는 생성된 아이콘을 터치하여 전체화면 앱 모드로 실행합니다.

### Android (Galaxy 등) 사용자
1. **Chrome 브라우저**를 열고 [수면 앱 바로 가기](https://rbtjd215.github.io/Deep_Sleep/)로 이동합니다.
2. 접속 직후 브라우저 하단에 뜨는 **'홈 화면에 Deep_Sleep_app 추가'** 팝업을 터치하거나, 우측 상단의 메뉴(점 3개) 버튼을 눌러 **'홈 화면에 추가'**를 선택합니다.
3. 홈 화면에 아이콘이 생성되면, 이를 터치하여 네이티브 앱처럼 전체화면으로 사용합니다.

---

## 2. 기본 사용 절차

1. **사운드 선택**: 앱 메인 화면 중앙의 옵션 버튼들을 눌러 원하는 수면 사운드(델타파, 세타파, 노이즈 등)를 선택합니다. 바이노럴 비트 효과를 온전히 경험하려면 **반드시 양쪽 이어폰(또는 헤드폰)을 착용**해야 합니다.
2. **호흡 횟수 설정**: 4-7-8 호흡 가이드의 사이클을 선택합니다.
   * **초보자 (4회)**: 약 1분 소요. 호흡법에 익숙하지 않은 경우 권장합니다.
   * **숙련자 (8회)**: 약 2분 소요. 스트레스가 심하거나 깊은 이완이 필요할 때 선택합니다.
3. **볼륨 조절**: 하단의 슬라이더를 이동하여 수면 사운드의 볼륨을 본인에게 가장 편안한 수준으로 맞춥니다. 너무 크면 수면에 방해가 될 수 있습니다.
4. **수면 모드 시작**: `수면 모드 시작` 버튼을 누르면 5초간의 준비 카운트다운 후 본격적인 호흡 가이드가 시작됩니다. 
5. **자동 전환 및 종료**: 호흡 가이드가 완료되면 화면이 수면 사운드 전용 화면으로 자동 전환되며, 30분 카운트다운이 작동합니다. 30분 후 사운드는 서서히 페이드 아웃되며 앱이 완전히 정지됩니다.

---

## 3. 호흡 가이드 상세 (4-7-8 호흡법)

호흡 가이드 단계에서는 화면 중앙의 구 형태(Orb)가 팽창 및 수축하는 애니메이션과 부드러운 차임벨 소리가 동기화되어 나타납니다. 이에 맞춰 아래 단계로 호흡을 통제합니다.

* **들이마시기 (4초)**: 입을 다물고 코를 통해 천천히 숨을 깊게 들이마십니다.
* **참기 (7초)**: 숨을 들이마신 상태에서 몸의 긴장을 풀고 7초간 호흡을 멈춥니다.
* **내쉬기 (8초)**: 입술을 오므리고 '후' 소리를 내며 8초 동안 몸 안의 공기를 모두 내뱉습니다.

이 과정을 설정한 횟수만큼 반복하면 부교감 신경계가 활성화되어 심박수가 안정되고 수면에 적합한 상태가 됩니다.

## 4. 지원되는 수면 사운드 종류

### 4.1. 바이노럴 비트 (Binaural Beats)
좌우 귀에 미세하게 주파수가 다른 소리를 각각 들려주어 뇌가 그 차이(Beat)에 해당하는 주파수에 동조하게 만드는 과학적 사운드 테라피입니다. 좌우 채널 분리가 필수적이므로 **이어폰 착용이 강제**됩니다.

* **델타 1Hz (깊은 수면)**: 수면의 가장 깊은 단계(서파 수면)에서 주로 발생하는 1Hz 델타파를 유도하여 육체적 회복을 촉진합니다.
* **델타 3Hz (수면 유도)**: 얕은 수면에서 깊은 수면으로 진입하는 경계 영역인 3Hz 주파수를 유도하여 빠르게 잠자리에 들 수 있도록 돕습니다.
* **세타 6Hz (깊은 이완)**: 명상 상태나 가벼운 수면 상태에서 발생하는 6Hz 세타파를 유도하여 정신적인 스트레스 해소와 릴랙스를 제공합니다.

### 4.2. 백색 소음 (Noise)
넓은 주파수 대역의 일정한 소음으로 주변의 돌발적인 잡음을 덮는(마스킹) 효과를 제공합니다. 이어폰 없이 **스피커로 재생해도 무방**합니다.

* **핑크 노이즈 (빗소리 느낌)**: 고주파 대역으로 갈수록 에너지가 자연스럽게 감소하여 사람의 귀에 가장 편안하게 들리는 소리입니다. 자연의 소리와 유사하여 수면 중 뇌파를 안정화합니다.
* **브라운 노이즈 (깊은 울림)**: 핑크 노이즈보다 저주파수(베이스) 영역이 훨씬 강하게 강조된 소리입니다. 묵직하고 폭포수 같은 깊은 울림을 주어 극심한 피로 상태에서 심리적 안정감을 줍니다.

## 5. 백그라운드 재생 및 기기 설정 주의사항

본 애플리케이션은 화면을 끄거나 잠금 화면으로 전환되어도 수면 사운드가 끊기지 않도록 백그라운드 무음 오디오 해킹 기법이 적용되어 있습니다. 사용 시 아래 사항을 확인하시기 바랍니다.

* **기기 무음 스위치 (iPhone)**: 기기 측면의 물리적인 무음 스위치가 켜져 있거나 '방해 금지 모드' 설정에 따라 브라우저의 미디어 소리가 출력되지 않을 수 있습니다. 이 경우 제어 센터를 열어 미디어 볼륨 슬라이더를 직접 올리거나 무음 설정을 해제해야 합니다.
* **화면 켜짐 유지 (Wake Lock)**: 호흡 가이드가 진행되는 동안에는 화면이 자동으로 꺼지지 않도록 제어됩니다. 따라서 호흡에 집중하기 위해 굳이 화면을 터치할 필요가 없습니다.
* **잠금 화면 제어**: 앱이 실행 중인 상태에서 화면을 끄면 스마트폰의 잠금 화면에 미디어 컨트롤러가 나타납니다. 앱을 켜지 않고도 잠금 화면에서 현재 오디오의 재생/일시정지 상태를 제어할 수 있습니다.
